## Key Metrics

### Application

| Metric | Warning | Critical |
|--------|---------|----------|
| Response time p95 | > 500ms | > 2000ms |
| Error rate | > 1% | > 5% |
| Request rate drop | > 20% | > 50% |
| 5xx responses | > 0.5% | > 2% |

### Infrastructure

| Metric | Warning | Critical |
|--------|---------|----------|
| CPU usage | > 70% | > 90% |
| Memory usage | > 80% | > 95% |
| Disk usage | > 70% | > 90% |
| Network errors | > 0.1% | > 1% |

### Database

| Metric | Warning | Critical |
|--------|---------|----------|
| Query time p95 | > 100ms | > 500ms |
| Connection pool usage | > 70% | > 90% |
| Deadlocks | > 0 | > 5/min |
| Replication lag | > 1s | > 10s |
